untuk belajar html kita harus mengetahui tentang struktur pada html.
struktur html terdiri dari 9 baris.yaitu:


berfungsi untuk menunjukkan ke komputer bahwa kita ingin membuat sebuah document html.


berfungsi untuk pembuka code html yang ingin kita masukkan.


berfungsi sebagai bagian kepala/head dari html.


digunakan untuk codingan awal dalam menambahkan judul,dan setelah menambahkan coding ini maka masukkan judul yang ingin anda tambahkan.


digunakan untuk mengakhiri input dari title.


berfungsi sebagai penutup dari kepala html.


codinagn ini adalah pembuka badan yang ada di html,dan setelah menambahkan codingan ini anda
dapat memasukkan codingan lainya yang ingin anda masukkan.


codingan ini berfungsi sebagai penutup dari body/badan yang telah kita buat tadi.


coding ini berfungsi sebagai penanda bahwa html yang anda buat telah selesai.
